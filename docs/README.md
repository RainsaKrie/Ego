## 当前不建议优先看的方式

不建议再从仓库根 `README` 里按完整清单逐个点读。

更好的顺序是：

1. 先看本页
2. 再读 A 类必读
3. 进入具体模块时再查 B 类
4. 只在需要回看历史或维护文档时再看 C 类

这样能明显减少“文档太多，但不知道先看什么”的负担。
